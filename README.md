# Config Files

My personal dotfiles and config files I sync across multiple machines.

| File | Description |
|------|-------------|
| [`~/.gitconfig`](.gitconfig) | Git |
| [`~/.vimrc`](.vimrc) | Vim |
| [`~/.config/nvim/init.lua`](.config/nvim/init.lua) | Neovim |
| [`keybindings.json`](keybindings.json) | VSCode keyboard shortcuts inspired by LazyVim motions with my own tweaks. Highly recommended to combine it with vim motions extension. |
| [`~/.tmux.conf`](.tmux.conf) | Tmux |
| [`~/.alacritty.toml`](.alacritty.toml) | Alacritty |

Note: Install TPM (Tmux Plugin Manager) before using the provided [`~/.tmux.conf`](.tmux.conf): https://github.com/tmux-plugins/tpm
